# Tool Upgrade Roadmap

## Rule
Upgrade only after clear revenue validation. Replace $0 placeholder tools with the minimum viable paid or self-hosted tool that saves time or protects revenue.

## Phase 1 — Validating ($0)
- Manual outreach from existing email/phone
- Free Gmail
- CSV/Google Sheets tracking
- Guided call console
- Manual scheduling
- Manual follow-up emails

## Phase 2 — First Revenue
Trigger: any $ from paying clients
Upgrades:
- Professional domain email address (~$12/year)
- Simple landing page or booking link
- Shared inbox or helpdesk if volume grows

## Phase 3 — $1k/mo for 3 months / business
Trigger: $1k MRR for 3 consecutive months for any single business
Decisions:
- Upgrade call tracking
- Upgrade CRM
- Upgrade scheduling system

## Phase 4 — Profitability Scaling
Trigger: multiple stable businesses and sustained profit
Decisions:
- Evaluate low-cost subscriptions only if ROI positive
- Upgrade automation
