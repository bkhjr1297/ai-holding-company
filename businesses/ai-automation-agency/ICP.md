# Ideal Customer Profile

## Primary ICP
- 5–50 employees
- Already paying for SaaS tools (CRM, support, scheduling, email)
- Revenue $500K–$20M/year
- Pain: leads falling through cracks, support backlog, manual busywork, inconsistent follow-up

## Best-fit Industries
1. Trades: HVAC, plumbing, electrical, roofing, cleaning
2. Professional services: accounting, legal, consulting, coaching
3. Local services: property management, fitness, clinics, salons
4. E-commerce / DTC: support tickets, returns, post-purchase messaging
5. Agencies / B2B services: lead response, onboarding, reporting

## Buyer Roles
- Founder / owner operator
- Operations manager / COO
- Marketing director
- Customer success / support lead

## Qualification Signals
- Active website form but slow/no reply
- Missed calls / unreturned voicemails
- Recurring “we should automate that” mentions
- Paying for tools but still doing work manually
