# Templates Pack

---

## 1. Episode Brief Template

```markdown
# EPISODE BRIEF

**Episode #:** {auto}
**Show:** {show_name}
**Guest name(s):** {name}
**Topic / theme:** {1-sentence}
**Proposed title:** {title}
**Proposed chapter markers:**
1. {timestamp} — {title}
2. {timestamp} — {title}
3. {timestamp} — {title}

**Prep materials provided:**
- [ ] Guest bio + photo
- [ ] Prior coverage / citations
- [ ] Sponsor reads / disclosure
- [ ] Key messages

**Special cues:**
- Bleep words: {list or "none"}
- Sponsor reads at: {timestamps}
- Music cues: {intro + outro cues}
```

## 2. Show Notes Template

```markdown
# Show Notes — {episode_number}: {episode_title}

**Audio link:** {master_mp3_url} **Duration:** {mm:ss}

## Overview
{150-200 word synopsis}

## Chapters
| Time | Title |
|---|---|
| 00:00 | Intro + cold open |
| {mm:ss} | {chapter_title} |
| ... | ... |

## Guest Bio
{2-3 sentence bio, linked to relevant social}

## Resources Mentioned
- {url/tool} — {1-line description}
- ...

## Quotes
> "{notable quote}" — {guest/time}

## Credits
Produced by Agency Agents for {show}. Mastered to -16 LUFS.
```

## 3. Spreadsheet Template (Episode Tracker)

- Episode number
- Title
- Status (Uploaded → Editing → QA → Show Notes → Publish → Confirmed)
- Raw file link
- Master file link
- Show notes link
- Transcript link
- Scheduled publish date
- Analytics report link
- Notes + revision log

## 4. Client Update Email Template

Subject: Episode {N} shipped — ready for review

```
Hi {client_first_name},

- Show notes: {link}
- Master MP3: {link}
- Chapter markers: included
- Loudness report: attached (-16 LUFS confirmed)

Please review by {deadline}. For anything you'd like changed,
reply directly to this thread and tag the edits. One revision
round included; we will consolidate feedback once before shipping.

If approved, we'll publish by your scheduled date ({date}).

—
{editor_name}
Agency Agents
```
