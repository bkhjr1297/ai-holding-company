# Acquisition System — AI Search Readiness Retrofit

Goal: get customers safely without unauthorized sending/calling.

## Funnel
1. Find public local-service websites.
2. Generate an AI Search Readiness Snapshot from public pages.
3. Create manual approval outreach draft.
4. Brian or an approved human reviews and sends only if appropriate.
5. Track reply/discovery/proposal/closed-won in CRM.
6. Use finished snapshots as proof samples.

## Automation allowed now
- Public website discovery.
- Public website scoring.
- Report generation.
- Draft generation.
- CRM updates.
- Proposal generation.
- RPG quest updates.

## Not automated yet
- No cold email sending.
- No calling.
- No contact form submissions.
- No paid tools.
- No bypassing blocks or scraping protected sources.
